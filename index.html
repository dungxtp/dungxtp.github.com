how to?

How to create express app + ejs

  1. npm init

  2. npm install express ejs –save

  3. mkdir views

  4. touch views/landing.ejs

  5. touch app.js

How to add CSS in express app

  1. mkdir public

  2. touch public/style.css

  3. inside app.js

  4. inside landing.ejs

How to create “partial”

  1. mkdir views/partials

  2. touch views/partials/header.ejs

  3. touch views/partials/footer.ejs

  4. landing.ejs

How to use body-parser

  1. npm install body-parser

  2. insert app.js

  3. touch views/new.ejs

how to use mongoose

  1. npm install mongoose –save

  2. touch app.js

2 New /dogs/new GET Display form to make a new dog
3 Create /dogs POST Add new dog to DB
4 Show /dogs/:id GET show info about one dog
5 Edit /dogs/:id/edit GET Show edit form for one dog
6 Update /dogs/:id PUT Update a particular dog, then redirect somewhere
7 Destroy /dogs/:id DELETE Delete a particular dog, the redirect somewhere
