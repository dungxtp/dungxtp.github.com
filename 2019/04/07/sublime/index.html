giải thích

click: bấm chuột trái
right click: bẫm chuột phải
double click: nháy chuột trái
hold ctrl: giữ ctrl
drag: rê chuột
arrow: 4 phím mũi tên
indent: thụt lề

đóng tab c

ctrl + w

tạo nhiều con trỏ

ctrl + click

bôi đen tất cả các từ giống nhau

alt + F3
or ctrl + d nhiều lần

bôi đen 1 dòng

ctrl + l

tạo nhiều con trỏ theo chiều dọc

shift + right click drag

thêm các con trỏ vào cuối dòng

bôi đen then ctrl + shift + L
or đặt con trỏ ở ngoài cùng bên phải then shift + right click drag
(then left arrow: các con trỏ ở đầu dòng)

alt + shift + w
// tạo thêm các con trỏ nếu muốn làm nhiều dòng

Mở log sublime text

// mở command
ctrl + `
// open log type this:
sublime.log_commands(True)
// stop log, nhấn arrow up(phím mũi tên lên trên) để hiện lại text open log
sublime.log_commands(False)

Macros

// mở và tắt ghi macro
ctrl + q
// save macro
Tools > Save Macro..

Key Bindings-tổ hợp phím (Preferences > Key Bindings)

// tạo phím tắt cho reindent
{“keys”: [“ctrl+shift+i”], “command”: “reindent”}
note: để tìm key như “reindent”: mở log rồi kích vào.
//thêm phím tắt di chuyển con trỏ lên đầu hoặc xuống cuối dòng
{ “keys”: [“ctrl+left”], “command”: “move_to”, “args”: {“to”: “bol”, “extend”: false} },
{ “keys”: [“ctrl+right”], “command”: “move_to”, “args”: {“to”: “eol”, “extend”: false} } note: ctrl + left(right) arrow
// tạo shortcut để tạo phím tắt cho macro, abc.sublime-macro: là tên của macro đã tạo
{“keys”: [“alt+enter”], “command”: “run_macro_file”, “args”: {“file”: “Packages/User/abc.sublime-macro”}}

Settings (Preferences > Settings)

//vintage mode (https://www.sublimetext.com/docs/2/vintage.html)
“ignored_packages”: [],
“vintage_start_in_command_mode”: true
// tô sáng dòng thêm mã sau vào settings
“highlight_line”: true,
// chiều rộng con trỏ
“caret_extra_width”: 2,
// chiều dài từ trên xuống
“caret_extra_top”: 2,
// chiều dài từ dưới lên
“caret_extra_bottom”: 2,
//chỉnh kích thước của dòng

“line_padding_bottom”: 1,
// in đậm tên folder
“bold_folder_labels”: true,
